# 🌍 Adventure Parallax Website

A visually engaging **Adventure-themed Parallax Website** built using **HTML and CSS**. This project creates an immersive scrolling experience with stunning parallax effects showcasing thrilling outdoor activities like **biking, paragliding, and surfing**.

## ✨ Features
- Smooth parallax scrolling effect
- Adventure-themed responsive design
- Sections for:
  - 🚴 Biking
  - 🪂 Paragliding
  - 🏄 Surfing
- Clean and modern UI
- Built with pure HTML & CSS

## 🛠️ Technologies Used
- HTML5
- CSS3
